## Enabling GitHub Pages

To enable GitHub Pages for this repository:

1. Go to your repository on GitHub
2. Click **Settings**
3. Navigate to **Pages** (in the sidebar)
4. Under "Source", select:
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /docs
5. Click **Save**
6. Your site will be published at `https://[your-username].github.io/finops-hub-deployment/`

## Local Development

To view the documentation locally:

1. Open `index.html` in your web browser
2. Or use a simple HTTP server:
   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js (with http-server)
   npx http-server
   ```

## Customization

The documentation is fully customizable:

- Edit `index.html` to update content
- Modify CSS variables in the `<style>` section to change colors
- Update subscription links and configuration values
- Add additional pages as needed
